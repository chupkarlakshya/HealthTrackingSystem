# Problem Statement
Users need a simple way to record daily health details like steps, calories, and workout time without using complex fitness apps or online services. HealthTrackingSystem solves this by offering a basic, offline, easy-to-use Tkinter application for daily health logging.

# Scope of the Project
- Desktop Python Tkinter application.
- Input: date, steps, calories, workout minutes.
- Store data in a text file.
- View all records or last 30 records.
- Show motivational messages.

# Target Users
- Students learning Python and GUI.
- Beginners wanting a simple health tracker.
- Users who prefer offline tools.
- Anyone needing basic daily health logging.

# High-Level Features
- Add health entries.
- View all saved entries.
- View last 30 (monthly) entries.
- Display motivational messages.
- Lightweight text-file-based storage.
- Simple and clean GUI.
